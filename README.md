# 🎨 SURSTA - Real-time Collaborative Whiteboard

A powerful, secure, and real-time collaborative drawing platform built for teams, educators, and creative minds. Draw, brainstorm, and share ideas instantly with password-protected rooms.

![SURSTA Demo](https://via.placeholder.com/800x400?text=Project+Screenshot+Placeholder) 

## 🚀 Features

- **Real-time Collaboration:** Multiple users can draw on the same canvas simultaneously using Socket.io.
- **Secure Room System:** Create private rooms with **Password Protection** to keep your drawings safe.
- **Advanced Tools:** - Brush, Eraser, and Shapes (Circle, Rectangle, Line).
  - Undo/Redo functionality.
  - Image Upload to annotate over pictures.
- **Unique Room URLs:** Share a simple link to invite others to your workspace.
- **Responsive UI:** Dark-themed, modern interface built with Tailwind CSS.

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite, Tailwind CSS, HTML5 Canvas API.
- **Backend:** Node.js, Express.js.
- **Real-time Engine:** Socket.io (WebSockets).
- **Deployment:** Vercel (Frontend) & Render/Railway (Backend).
